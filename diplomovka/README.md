1) Požiadavky
- Python 3.10 – 3.12 (odporúčané)
- Visual Studio Code alebo iné IDE podporujúce Python

2) Vytvorenie a aktivácia virtuálneho prostredia (venv)
- vo Windows (PowerShell) spustite nasledujúce príkazy:
python -m venv .venv
.\.venv\Scripts\Activate.ps1

3) Inštalácia závislostí
pip install --upgrade pip
pip install -r requirements.txt

4) Spustenie Experimentu 1
python experiment_1.py
